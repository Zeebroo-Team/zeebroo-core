# Zeebroo marketing site (static HTML)

Plain HTML/CSS/JS for business owners — **no npm, yarn, or build tools**.

## Preview

```bash
cd website_html_pages
python3 -m http.server 8080
# http://localhost:8080
```

## Contents

- Benefits-focused copy (growth, clarity, daily operations)
- Solutions: stock, POS, HR, social campaigns, accounts, bills, ledgers, desktop POS (Windows, Linux, macOS)
- Hero: copy on the left, product video on the right (`assets/demo.mp4`)
- **Everything in one solution** section: interactive ASCII corridor animation (same style as app sign-in)

## Deploy

Upload the `website_html_pages/` folder to any static web host.
